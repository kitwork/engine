Routing & Config

Kitwork uses a high-performance Radix Tree router written natively in Go. This allows static routing, parameter binding, and catch-all logic with zero overhead.

Static Endpoints

Routes are mapped via the underlying fast Go router. Define standard HTTP verbs like get() or post().

router.get("/about").handle((request, response) => { return response.text("Kitwork is ready."); });

Dynamic Parameters

You can extract path parameters safely using :param syntax inside the URL declaration.

router.get("/users/:id").handle((request, response) => { // Fetch user parameter from the request object const uid = request.params("id"); return response.json({ user_id: uid }); });

Template Rendering

Return fully compiled HTML templates using the Go template engine.

const home = render("/pages/home").layout("/layouts/home"); router.get("/").handle((request, response) => { const page = home.page("index.html"); return response.html(page.bind({ title: "Dashboard" })); });
