Kitwork Docs

Everything you need to know to build blazing fast web applications using the Sovereign Stack-Based Engine.

1. Routing & Handlers

Kitwork uses an internal Radix Tree to map routes with zero latency. Define routes directly via the `router` object in your `app.js` file.

const { router } = kitwork();

// Basic GET route passing a context mapping
router.get("/hello").handle((request, response) => {
    return response.text("Hello World!");
});

2. Database ORM

Connect to Postgres, MySQL, or SQLite natively. Our chainable API allows you to fetch, update, and delete without writing raw SQL.

// Fetch user by ID efficiently
const user = db.table("user").where(u => u.id == 5).first();

// List exactly 10 active users without boilerplate
const activeUsers = db.table("user").limit(10).list(u => u.is_active == true);

3. Goroutines in JS (True Async)

Need to send an email, process an image, or write logs? Push it to the background instantly using `go()`. Unlike NodeJS, the main thread will never be blocked.

go(() => {
    // This lambda executes automatically in a parallel OS thread!
    log.Print("Background task is deeply processing...");
});
