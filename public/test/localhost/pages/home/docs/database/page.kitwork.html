Database ORM

Kitwork features a built-in natively compiled Query Builder that avoids raw SQL entirely while running at maximum CGO speeds.

Retrieving Data

Chain filters safely without SQL Injection vulnerabilities.

// Select Multiple Rows const users = db.table("user").limit(50).list(); // Filter Single Row using Where Condition const admin = db.table("user").where(u => u.role == "admin").first();

Inserting & Updating

Create or update records dynamically using pure JSON mapping objects.

db.table("user").create({ username: "hacker101", email: "test@kitwork.io", is_active: true }); db.table("user").where(u => u.id == 5).update({ is_active: false });
