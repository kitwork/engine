app.kitwork.js
const { router, log, database, go } = kitwork();

const db = database.connect({ type: "postgres" });

router.get("/users/:id?").handle((req, res) => {
    // 1. Database query via ORM
    const users = db.user.limit(5).list();

    // 2. Spin up parallel Goroutine
    go(() => {
        log.Print("⚡ Isolated background sync");
    });

    return res.json({ data: users });
});

The Energy Economy

Your code pays for
its own existence.

Spam is solved by physics. Kitwork tracks the Energy Cost of every operation within the VM. When logic exceeds its allocated energy budget, it is throttled or halted. Efficiency is the currency of the engine.

  • C CPU (Calc) | Thinking is cheap.
  • R RAM (Alloc) | Memory is finite space.
  • N Network | Communication is expensive.
scheduler.js
const { scheduler } = kitwork();

// Natural Language Scheduling
scheduler
  .daily("01:00")
  .every("10s")
  .handle(() => {
    // Logic executed within energy bounds
    database.logs.where(l => l.age > "30d").destroy();
  });
