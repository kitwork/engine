Database ORM

Kitwork features a built-in natively compiled Query Builder that avoids raw SQL entirely while running at maximum CGO speeds.

Retrieving Data

Chain filters safely without SQL Injection vulnerabilities. The engine automatically handles parameter binding.

// Select Multiple Rows
const users = db.table("user").limit(50).list();

// Filter Single Row
const admin = db.table("user").where(u => u.role == "admin").first();

Create & Update

Manipulate data using pure JSON objects. Validation is performed at the engine level.

db.table("user").create({
    username: "hacker101",
    email: "test@kitwork.io",
    is_active: true
});

db.table("user").where(u => u.id == 5).update({
    is_active: false
});

Deleting Data

Permanently remove records from the database using the same chainable syntax.

db.table("sessions").where(s => s.expired == true).delete();
