## 2. ‚ö° The Syntax Strategy

We emphasize **Explicit Precision**. Standard units are pixels (`32px`), percentages (`100%`), or keywords (`block`).

### üìê Sizing & Spacing
Use `{property}-{value}`.
*   **Width/Height**: `width-100%` (or `width-full`), `height-screen` (`100vh`), `width-320px`.
*   **Margin/Padding**: `margin-32px`, `padding-x-64px`, `margin-bottom-12px`.
*   **Gap**: `gap-24px`, `gap-x-12px`.

### üî† Typography
*   **Font Family**: `font-outfit` (UI), `font-mono` (Code).
*   **Size**: `font-size-16px`, `font-size-72px`.
*   **Weight**: `font-bold`, `font-light`, `font-black`.
*   **Alignment**: `text-center`, `text-right`.
*   **Transform**: `uppercase`, `lowercase`, `italic`, `underline`.

### üé® Colors & Effects
*   **Text**: `text-white`, `text-brand`, `text-white-50` (50% opacity).
*   **Background**: `background-black`, `background-dark`, `background-brand`.
*   **Border**: `border-white-10`, `border-brand-20`.
*   **Shadow**: `shadow-small`, `shadow-glow-brand` (Neon effect), `shadow-system` (Deep depth).

## 3. üì± Responsive Design

We use a **Desktop-First** approach with explicit override prefixes.

| Prefix | Breakpoint | Description |
| :--- | :--- | :--- |
| *(none)* | `All` | Default style (Desktop/All) |
| **`laptop:`** | `< 1200px` | Laptop screens |
| **`tablet:`** | `< 900px` | Tableau / Portrait |
| **`mobile:`** | `< 600px` | Mobile Phones |

**Example:**
```html
<div class="width-50pct tablet:width-100pct mobile:display-none">
  <!-- 
    - 50% width normally.
    - 100% width on Tablet and smaller.
    - Hidden completely on Mobile.
  -->
</div>
```

## 4. üñ±Ô∏è Interaction & States

Apply styles on specific states using state prefixes. You can **stack** them with responsive prefixes!

| Prefix | Trigger | Example |
| :--- | :--- | :--- |
| **`hover:`** | Mouse over | `hover:text-brand` |
| **`focus:`** | Input focus | `focus:border-brand` |
| **`group-hover:`** | Parent hovered | `group-hover:opacity-100` |

**Example:**
```html
<button class="background-dark hover:background-brand transition-colors duration-300">
  Click Me
</button>
```

**Advanced Stacking:**
```html
<div class="opacity-0 tablet:hover:opacity-100">
  <!-- Visible only when hovering ON A TABLET (Odd use case, but supported!) -->
</div>
```

## 5. üß© Layout System (Flex & Grid)

### Flexbox
*   **Enable**: `display-flex`, `display-inline-flex`.
*   **Direction**: `flex-row`, `flex-column`.
*   **Alignment**: `items-center`, `justify-between`, `items-start`.
*   **Wrap**: `flex-wrap`.

### Grid System (12 Columns)
*   **Enable**: `display-grid`.
*   **Columns**: `grid-columns-12`, `grid-columns-2`, `grid-columns-1`.
*   **Spanning**: `grid-span-4` (Span 4 cols), `grid-span-full` (Span all).

**Example Grid Layout:**
```html
<div class="display-grid grid-columns-12 gap-24px">
  <div class="grid-span-8 tablet:grid-span-12">Main Content</div>
  <div class="grid-span-4 tablet:grid-span-12">Sidebar</div>
</div>
```
