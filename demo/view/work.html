Core Philosophy.

Nanosecond Execution

A proprietary stack-based VM that executes bytecode instructions in ~70ns. Zero reflection, zero overhead during logic runtime.

Zero-GC Pressure

The engine pools task contexts and VM stacks, ensuring consistent performance without GC pauses even under heavy concurrency.

Logic as Infrastructure

Decouples business rules from server implementation. Your logic becomes a portable asset, executed by a high-velocity core.

Agent-Native Design

Built for the future of AI. The engine’s bytecode and task context are designed to be easily manipulated by AI agents while remaining 100% predictable for humans.

Smart ORM.
Native Feel.

Kitwork leverages Proxy-based Entity Resolution to eliminate boilerplate. No more SQL strings; just pure, predictable logic.

  • Proxy Entity: Access tables as native properties.
  • Lambda Filters: Write natural JS conditions.
  • Auto IN: Batch lookup using array inclusion.
  • Strict Updates: Secure data writing by default.
orm_operations.js
// 1. Lambda-based Filtering
const active = db.user.find(u => u.email == "admin@kitwork.io");

// 2. Batch Lookup with Automatic IN

// 3. Strict Update (Secure)
